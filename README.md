Hosted via GitHub Pages.


## URLs (after GitHub Pages setup)

- `https://petukhovdn.github.io/app-legal/argumentator/privacy.html`
- `https://petukhovdn.github.io/app-legal/argumentator/terms.html`

## Setup

1. Go to repo Settings > Pages
2. Source: Deploy from a branch
3. Branch: main, folder: / (root)
4. Save — pages will be live in ~1 minute
