Politique de Confidentialité

Dernière mise à jour : 2 avril 2026
EN RU DE ES FR PT ZH

Quelles données nous collectons

Données du compte : Adresse e-mail et nom d'affichage que vous fournissez lors de l'inscription. Si vous vous connectez avec Google ou Apple, nous recevons votre nom et votre adresse e-mail de ces services.

Données d'utilisation : Nombre de requêtes effectuées, paramètres d'arguments sélectionnés (ton, stratégie, force), horodatages.

Historique : Vos arguments générés sont stockés sur nos serveurs afin que vous puissiez y accéder depuis n'importe quel appareil. Vous pouvez supprimer des éléments individuels ou l'intégralité de votre historique à tout moment.

Données de l'appareil : Type d'appareil, version du système d'exploitation, version de l'application — collectées automatiquement pour les rapports de plantage et les analyses.

Ce que nous ne collectons PAS

Nous ne collectons pas votre localisation, vos contacts, vos photos ni aucune donnée sans rapport avec les fonctionnalités de l'application.

Comment nous utilisons vos données

  • Pour fournir le service (générer des arguments en fonction de vos requêtes)
  • Pour appliquer les limites d'utilisation selon le niveau d'abonnement
  • Pour sauvegarder votre historique d'arguments et vos préférences
  • Pour améliorer le service (statistiques d'utilisation agrégées et anonymisées)
  • Pour envoyer des notifications push (défis quotidiens) si vous y consentez

Services tiers

  • Firebase (Google) : Authentification, rapports de plantage, analyses. Politique de Confidentialité
  • MongoDB Atlas : Hébergement de base de données. Politique de Confidentialité
  • RevenueCat : Gestion des abonnements. Politique de Confidentialité
  • Tavily : Recherche web pour les arguments sourcés. Politique de Confidentialité
  • AdMob (Google) : Publicité pour les utilisateurs du forfait gratuit. Politique de Confidentialité

Conservation des données

Les données du compte sont conservées jusqu'à la suppression de votre compte. L'historique des arguments est conservé pendant 5 ans ou jusqu'à ce que vous le supprimiez manuellement. Les compteurs d'utilisation sont réinitialisés quotidiennement. Les journaux serveur sont conservés pendant 30 jours.

Sécurité des données

Toutes les données sont transmises via HTTPS. Les mots de passe sont hachés à l'aide de bcrypt. L'accès à la base de données est limité aux seuls services autorisés.

Vos droits

Vous pouvez demander la suppression de votre compte et de toutes les données associées en nous contactant. Vous pouvez supprimer votre historique d'arguments directement dans l'application.

Enfants

Cette application n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données concernant les enfants de moins de 13 ans.

Modifications de cette politique

Nous pouvons mettre à jour cette politique de temps à autre. Nous vous informerons des modifications importantes via l'application.
